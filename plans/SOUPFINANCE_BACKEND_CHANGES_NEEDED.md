**Endpoint:** `GET /rest/ledgerAccount/balance/{id}.json`

```json
  "accountName": "Cash",
  "balance": 15000.00,
  "debitTotal": 50000.00,
  "creditTotal": 35000.00,
  "currency": "GHS"
}
```

**Test:** `e2e/integration/accounting.integration.spec.ts` - "can get account balance"
